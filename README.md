# Web Scraping Tutorials

This repository contains a collection of tutorials extracted data from the web using Python. It covers various tools and techniques, from basic HTML parsing to handling dynamic content.

## Table of Contents
* [Libraries Used](#libraries-used)

## Libraries Used
The scripts in this repository primarily use the following Python libraries:
* **BeautifulSoup4:** For parsing HTML and XML documents.
* **Requests:** For making HTTP requests to retrieve web pages.
* **Selenium:** For automating web browsers to scrape dynamic content.

## Installation
To run these tutorials locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/sarahjoy18/webscraping_tutorials.git](https://github.com/sarahjoy18/webscraping_tutorials.git)

2. Navigate to the directory:
   ```bash
   cd webscraping_tutorials

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
